What's an accordion?

An accordion always contains the category title, an expanded and a collapsed state, an icon indicating expansion, and the spacing between them.

When and how should it be used?

It should be used when users only need a few key concepts or descriptions of the content on a single page.

What happens if the user clicks on a collapsed card while another card is open?

It happens that the open card was closed, to give way to the information of the next open card, but there are different designs that prefer it the other way around.

How to choose an icon to indicate expansion?

You must choose a different icon to open and close, so that the user understands what action he took.

Can an accordion contain other elements?

Yes, accordions can contain text, images, videos, or even other accordions for deeper navigation.

Should accordions be used for FAQs?

Yes, they are commonly used for FAQs because they help organize content efficiently.

Are accordions good for mobile design?

Yes, accordions help reduce clutter on small screens by allowing users to expand content only when needed.

How many items should an accordion have?

It depends on the use case, but it's best to limit it to around 5-7 items for readability.

Can accordions improve page load speed?

Yes, since accordions hide unnecessary content initially, they can help with performance by reducing the initial load time.
